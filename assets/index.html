Meu nome é Matheus Link, Desenvolvedor Front-End, ainda sem experiência de trabalho, porém quero compartilhar conhecimentos que eu adquiri até aqui na minha jornada, e nessa página eu separei um guia básico do que você deve fazer durante o começo de sua caminhada nesse mundo que não para !

1. PRECISO TER UM EXCELENTE NOTEBOOK / PC PARA COMEÇAR ?

Não! Para o começo de seus estudos você não precisa ter uma super-configuração em sua máquina, com um processador de 2.0 GHz, 4gb de memória RAM e pelo menos 128 GB de armazenamento interno você já pode começar. Será um desempenho aceitável para estudos, porém será quase impossível de se conseguir trabalhar com uma configuração parecida quando você conseguir uma vaga.


2. PRECISO SER UM GÊNIO NA MATEMÁTICA ?

NÃO! Você não precisa ser um gênio na matemática para ser um programador, mas é bom saber pelo menos o básico dela, sempre é importante!


3. COMO COMEÇAR A ESTUDAR ?

Muitos têm dúvidas se é necessário comprar um curso caro, ou assinar alguma plataforma para estudar, a resposta é NÃO, você pode começar a estudar por uma plataforma mundialmente conhecida chamada YouTube, onde existem excelentes professores, eu por exemplo, comecei estudando com o professor Gustavo Guanabara, do canal Curso em Vídeo.

Você também pode, e deve dar umas lidas no material da MDN, onde temos tudo sobre o mundo da programação, ainda existem diversos outros sites, com blogs e muitos materiais sobre esse mundo!


4. O QUE DEVO ESTUDAR ?

No Front-End temos 3 principais tecnologias, o HTML, o CSS e o JavaScript.

Mas comece estudando Lógica de Programação. Será muito importante, é o pilar para desenvlover seus estudos, é como aprender as palavras de um idioma, palavras que no futuro irão se tornar frases, textos e afins.


HTML (HyperText Markup Language), que não é uma linguagem de programação, mas sim de marcação, com ele você vai entender como as páginas da Web são montadas. Ele é o conteúdo da página.

Durante seus estudos de HTML você já será introduzido ao CSS (Cascading Style Sheets), que é uma Linguagem de Estilização, é uma ferramenta que muda as cores, tamanhos, algumas animações e fontes de elementos do HTML.


Quando você já tiver um bom entendimento dessas duas, crie um projeto simples com seus conhecimentos, como um mini blog ou um exemplo de layout para um perfil.

Após isso você pode partir para o JavaScript, que aí sim é uma linguagem de programação, com ele você poderá adicionar funcionalidades ao seu projeto, como uma validação de um formulário, a exibição de uma janela Pop-up na tela, e muitas outras funcionalidades. Ele é uma linguagem versátil que desempenha um papel fundamental na criação de interatividade e dinamismo nas páginas da web, tornando a experiência online mais rica e envolvente para os usuários.

O JavaScript também pode ser usado no Back-End, e não confunda JavaScript com Java, são duas linguagens muito diferentes.


Estarei deixando a seguir um guia do que se estudar para se ter um bom entendimento em cada uma dessa ferramentas.

HTML

  • -Estrutura Básica: Aprenda a estrutura básica de um documento HTML, incluindo as tags html, head, title e body.
  • -Tags e Elementos: Familiarize-se com as tags HTML mais comuns, como p, h1, ul, li, a, img, etc, e aprenda como usá-las corretamente.
  • -Formulários: Estude como criar formulários usando tags como form, input, textarea, select, e saiba como processar os dados enviados por formulários.
  • -Links e Âncoras: Aprenda a criar links usando a tag a, entender atributos como href, target, e como vincular a outras páginas.
  • -Semântica: Explore as tags semânticas, como header, nav, main, article, footer, que dão significado estrutural ao conteúdo.

CSS

  • -Seletores e Propriedades: Entenda como usar seletores CSS para aplicar estilos a elementos HTML e como definir propriedades como color, font-size, background, etc.
  • -Box Model: Aprenda sobre o modelo de caixa, que inclui margens, bordas, preenchimento (padding) e dimensões (width e height) dos elementos.
  • -Posicionamento e Layout: Estude diferentes técnicas de posicionamento, como static, relative, absolute e float, além de como criar layouts usando flexbox e grid.
  • -Tipografia e Cores: Explore como controlar a aparência do texto (fontes, estilos, alinhamento) e como usar cores e gradientes nos elementos.
  • -Media Queries e Responsividade: Saiba como criar designs responsivos usando media queries para adaptar o layout a diferentes tamanhos de tela.

JavaScript

  • -Sintaxe e Variáveis: Aprenda a sintaxe básica do JavaScript, incluindo declaração de variáveis, tipos de dados (strings, números, booleanos), operadores, etc.
  • -Estruturas de Controle: Estude estruturas como if, else, switch, loops for e while para controle de fluxo de programa.
  • -Funções: Entenda como criar e chamar funções, passar argumentos e retornar valores, além de explorar escopo de variáveis.
  • -Manipulação do DOM: Saiba como interagir com o Document Object Model (DOM) para alterar elementos HTML, criar e remover elementos dinamicamente.
  • -Eventos: Aprenda a lidar com eventos, como cliques de mouse, pressionamento de teclas, e como associar funções a esses eventos.
  • -AJAX e Integração com APIs: Explore como fazer chamadas assíncronas à web usando AJAX para carregar e enviar dados de e para servidores.
  • -Introdução a Frameworks: Se desejar, dê uma olhada em bibliotecas e frameworks populares como jQuery, React ou Vue.js, que simplificam e aceleram o desenvolvimento.

Você também vai precisar aprender sobre GIT e GitHub

GIT: O Git é como uma máquina do tempo para o seu código. Ele registra todas as alterações que você faz, permitindo que você veja como o projeto evoluiu ao longo do tempo. Isso é inestimável, pois garante que você nunca perca uma ideia ou possa sempre voltar a um estado anterior se algo der errado. Imagine poder explorar diferentes abordagens e experimentar mudanças sabendo que você pode voltar atrás se necessário. O versionamento de código, proporcionado pelo Git, é uma forma de preservar suas ideias e conquistas, além de aprender com os erros.


GitHub: O GitHub é o lugar onde você pode guardar o seu código online. Por que isso é importante? Imagine que você está trabalhando em seu projeto pessoal e deseja compartilhá-lo com outras pessoas para receber feedback ou até mesmo ajuda. O GitHub facilita a colaboração, permitindo que outras pessoas vejam, copiem (fork) seu projeto para fazer suas próprias melhorias e sugestões (pull requests). Isso é especialmente poderoso para projetos de código aberto, onde você pode contribuir para outros projetos e aprender com os desenvolvedores experientes.
